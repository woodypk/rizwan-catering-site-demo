# Rizwan Pakwan — Catering & fast food (demo site)

Static marketing site for **Rizwan Pakwan BBQ & Fast Food**: menu with rates, WhatsApp ordering, event catering (corporate / wedding / religious), location with map, and order CTAs.

## Live preview

After GitHub Pages is enabled, your site will be at:

**`https://woodypk.github.io/rizwan-catering-site-demo/`**

If your files live inside a **`cateting`** folder in the repo, either:

- move **`index.html`** and **`assets/`** to the **repository root**, **or**
- in **Settings → Pages**, set the publishing folder to **`/cateting`** (if that option matches your layout).

Until the homepage is at the repo root (or Pages points at the right folder), the demo link may show a 404 or an empty page.

## Tech stack

- HTML5  
- CSS (custom layout, responsive sections)  
- Minimal JavaScript (carousel, mobile nav, scroll reveal, footer year)

## Run locally

1. Clone or download this repository.  
2. Open **`index.html`** in a browser, **or** serve the folder with any static server (e.g. VS Code “Live Server”).

Paths assume **`index.html`** at the project root next to **`assets/`**.
